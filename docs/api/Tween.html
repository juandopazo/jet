Class Tween - extends Base

A Tween is a variation of a property during a lapse of time that has a certain easing associated

Configuration Attributes

duration - {String | Number} Allowd strings: "fast", "slow", "normal". Numbers are milliseconds

The duration of the animation
Default Value: 1000

easing - Function

The easing used by the animation
Default Value: Easing.linear

easingStrength - Number

The strength of the easing if applicable. Must be >= 0
Default Value: 2

from - Object

an object literal with properties that will be animated

node - writeonce object

The node that will be animated

playing - Boolean

Whether the animation is playing or not

to - Object

an object literal with the target properties for the animation
