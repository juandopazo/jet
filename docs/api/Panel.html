Class Panel - extends Overlay

Known Subclasses:
SimpleDialog
A panel is an overlay that resembles an OS window without actually being one, to the problems they have (stop javascript execution, etc)

Constructor

Panel ( config )
Parameters:
config <Object> Object literal specifying widget configuration properties

Configuration Attributes

close - Boolean

If true, a close button is added to the panel that hides it when clicked
Default Value: true

contentBox - object

A panel uses another container inside the boundingBox in order to have a more complex design (ie: shadow)

shadow - object

If true, the panel shows a shadow
Default Value: true

underlay - object

The underlay is inserted after the contentBox to allow for a more complex design
