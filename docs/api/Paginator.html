Configuration Attributes

currentPage - writeonce Number

The current selected page. If set in the config, the paginator will go directly to that page when rendered

firstLast - Boolean

Wheter the "First" and "Last" buttons should appear
Default Value: true

firstText - writeonce String

The text of the "first" button
Default Value: "<< first"

lastText - writeonce String

The text of the "last" button
Default Value: "last >>"

nextText - writeonce String

The text of the "next" button
Default Value: "next >"

pageCount - Number

The number of pages in the paginator

pagesShown - Number

Number of pages that should be listed in the paginator (1, 2, 3...)
Default Value: 5

prevNext - Boolean

Wheter the "previous" and "next" buttons should appear
Default Value: true

prevText - writeonce String

The text of the "previous" button
Default Value: "< prev"

recordSet - RecordSet

A RecordSet with the data the paginator should handle

recordsPerPage - Number

Number of records each page should show
Default Value: 10
