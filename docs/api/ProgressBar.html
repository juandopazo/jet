Class ProgressBar

A progressbar for informing the user of the status of loading resources. Other uses include equalizers, ratings, etc

The value of the attribute, prior to it being set

Configuration Attributes

animate - writeonce object

Whether to animate the progressbar progress. The Anim module must be present
Default Value: false

bar - object

The DOM element that represents the bar in the progressbar

direction - writeonce object

Direction in which the progressbar increases its size. May be "ltr", "ttb" or "btt"
Default Value: "ltr"

duration - object

Duration of the animation in case it is being used
Default Value: 500 ms

easing - object

Easing to use when animating
Default Value: linear

maxMvalue - object

Maximum value that the progressbar data will take
Default Value: 100

minMvalue - object

Minimum value that the progressbar data will take
Default Value: 0

value - object

Current value the progressbar is taking
Default Value: 0
