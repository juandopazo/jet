Class Overlay

Known Subclasses:
Panel
An Overlay is a Module that floats in the page (doesn't have position static)

Constructor

Overlay ( config )
Parameters:
config <Object> Object literal specifying widget configuration properties

Configuration Attributes

bottom - Number

The bottom position in pixels

center - Boolean

If true, the overlay is positioned in the center of the page
Default Value: true

draggable - object

If true, the overlay can be dragged
Default Value: false

fixed - Boolean

If true, the overlay is position is set to fixed
Default Value: false

height - Number

The height of the overlay. If set to 0 (zero) the height changes with the content
Default Value: 0

left - Number

The left position in pixels

right - Number

The right position in pixels

top - Number

The top position in pixels

width - Number

The width of the overlay
Default Value: 300
