Class Module - extends Widget

Known Subclasses:
Overlay StaticPanel
A module is a basic container with header, body and footer

Constructor

Module ( config )
Parameters:
config <Object> Object literal specifying widget configuration properties

Configuration Attributes

body - writeonce object

A pointer to the body node

bodyContent - DOM Node | String | NodeList

The body of the module. If set to a string a node is creating and the string is set to its innerHTML A body is always present in a Module
Default Value: ""

footer - writeonce object

A pointer to the footer node

footerContent - DOM Node | String | NodeList

The footer of the module. If set to a string a node is creating and the string is set to its innerHTML

header - writeonce object

A pointer to the header node

headerContent - DOM Node | String | NodeList

The header of the module. If set to a string a node is creating and the string is set to its innerHTML
