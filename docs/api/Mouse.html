Class Mouse - extends Utility

A utility for tracking the mouse movement without crashing the browser rendering engine. Also allows for moving the mouse over iframes and other pesky elements

Constructor

Mouse ( config )
Parameters:
config <Object> Object literal specifying configuration properties

Methods

down

void down ( )
Start tracking. Equivalent to setting the tracking attribute to true. Simulates the mousedown event
Chainable: This method is chainable.

up

void up ( )
Stop tracking. Equivalent to setting the tracking attribute to false Simulates the mouseup event
Chainable: This method is chainable.

move

move ( )
Fires not when the mouse moves, but in an interval defined by the frequency attribute This way you can track the mouse position without breakin the browser's rendering engine because the native mousemove event fires too quickly
