Class EventTarget

Known Subclasses:
Attribute

A class designed to be inherited or augmented into other classes and provide custom events.

Custom events work by attaching event listeners to a class that augments or extends EventTarget. An event listener can be a function or an object with a method called "eventHandler". If it is a function, when fired the context will be the firing object. In the case of an object, the context will be the object itself.

Attaching an object to the "*" event type allows for a different approach:

someObj.handleEvent = function (e) {
    switch (e.type) {
        case "someEvent":
            //do something
            break;
        case "otherEvent":
            //do something else
            break;
    }
};
eventProvider.on("*", someObj);
