each

void each ( callback )
Iterates through the NodeList The callback is passed a reference to the node and an iteration index. The "this" keyword also refers to the node. ie:
$("div").each(function (node, i) {
    if (i % 2 == 1) {
        $(node).addClass("even");
    } else {
        $(node).addClass("odd");
    }
});
Parameters:
callback <Function>
Chainable: This method is chainable.
