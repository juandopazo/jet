Class LayoutPanelBase

Base class used to create both the Layout and LayoutPanel classes. Provides functionality to resize panels keeping the declared layout structure

Constructor

LayoutPanelBase ( )

Configuration Attributes

defaultChildType - object

Default type to apply to children
Default Value: LayoutPanel

direction - writeonce object

Direction in which this panel can be resized
Default Value: LayoutPanelBase.Vertical

minSize - writeonce object

Minimum size the panel can acquire. Applies to width or height depending on the 'direction' attribute
Default Value: 0

shim - writeonce object

Whether the resize utility should use a shim to protect the mouse movements
Default Value: false
