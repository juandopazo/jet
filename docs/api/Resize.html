Class Resize - extends Utility

Provides a utility for resizing elements

Properties

handles - Array

qconfig handles
Default Value: ['b', 'r', 'br']

reposition - Boolean

If set to true, the resize utility will automatically change the position of the node so that is stays in place when resizing it in any direction
Default Value: false

Configuration Attributes

animate - Boolean

Creates an animation when the resize handle is released. Can only be set to true if 'proxy' is set to true. Requires the Anim module.
Default Value: false

constrain - Boolean

If set to true, the node can't become bigger than the screen
Default Value: false

hiddenHandles - Boolean

If set to true, the handles are interactive but invisible
Default Value: false

locked - Boolean

If the resize is locked the handles are not interactive
Default Value: false

minHeight - Number

The minimum height the node can achieve
Default Value: 0

minWidth - Number

The minimum with the node can achieve
Default Value: 0

node - object

The node to be resized

shim - Boolean

Uses invisible elements to be able to resize the node over iframes
Default Value: false

useProxy - writeonce Boolean | String

Whether to use a copy of the node while resizing or not. Possible values: false, true, 'clone'
Default Value: false
